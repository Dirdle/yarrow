USAGE
---------
To use, compile and run from the command line with 6 arguments in quotes ("") separated by spaces:
-The location of the database
-The database's username
-The database's password
-The schema within the database that contains the relevant tables. This can be left blank, ie ""
-The location of the file containing the person data
-The location of the file containing the order data

Alternatively, import into an IDE such as NetBeans and set the run configuration to use the same arguments
